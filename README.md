# ms-strlib
C String library contains the following functions:
- ```ms_strlib_remove_backspaces()```
- ```ms_strlib_strtok()```
- ```ms_strlib_strdelch()```
- ```ms_strlib_toupper()```
- ```ms_strlib_tolower()```


# Build (Windows)
1. Clone project
```git clone --recursive git@github.com:Mrunmoy/ms-strlib.git```
2. Open in CLion
3. Select target as `strlib-unit-tests`
4. Build and Run
